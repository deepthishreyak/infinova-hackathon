## 🚀 HOW TO RUN RIGHT NOW

### ✅ Option 1: Docker (Easiest - 1 Command)

```bash
cd c:\Users\deept\OneDrive\Desktop\infinova-hackathon
docker-compose up
```

Then open browser: **http://localhost:3000**

**What Happens:**
- Automatically builds 3 services
- Sets up networking
- Starts all 3 services
- Ready to use

### ✅ Option 2: Manual (3 Terminals)

First, ensure you have:
- Node.js 16+ (from nodejs.org)
- Python 3.8+ (from python.org)

Then open **3 terminal windows** in project root:

**Terminal 1 - Backend:**
```bash
cd backend
npm install
npm start
```
Runs on: http://localhost:3001

**Terminal 2 - AI Service:**
```bash
cd ai-service
python -m venv venv
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn main:app --reload
```
Runs on: http://localhost:8000

**Terminal 3 - Frontend:**
```bash
cd frontend
npm install
npm start
```
Runs on: http://localhost:3000
